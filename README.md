# Content Team Tools

A collection of internal tools built for the RevvGrowth content team. This repo is where we experiment, build, and ship tools that make the content process faster and smarter.

---

## Tools

### [LLM Content Scorer](./scorer)
A web app that scores blog posts and articles for AI engine citation readiness. Paste a URL or raw text and get an instant 0–40 score across 29 criteria — with a grade, critical failure flags, and exact remediation steps.
